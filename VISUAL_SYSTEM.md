# VISUAL_SYSTEM.md

## Visual style
Use a clean, modern SaaS dashboard style.

Keywords:
- professional
- trustworthy
- minimal
- structured
- light
- spacious

## Layout
- desktop width
- card-based layout
- balanced white space
- consistent margins and padding
- clear alignment

## Color strategy
Use restrained semantic color:
- blue for neutral/informational
- yellow or amber for caution
- orange for warning
- red for action needed / errors
- gray neutrals for text and surfaces

Do not oversaturate the UI.

## Typography
Use clear sans-serif typography.
Prefer:
- strong page title
- clear KPI numerals
- muted supporting text
- readable labels and chart legends

## Components
Use consistent styling for:
- KPI cards
- buttons
- filter controls
- badges/status pills
- modal
- chart container
- navigation tabs

## Desired visual characteristics
- subtle shadows
- soft corners
- consistent badge shapes
- balanced spacing
- clean chart styling

## Avoid
- overly playful visuals
- neon colors
- heavy gradients everywhere
- glassmorphism excess
- cluttered tables
- unnecessary complexity
